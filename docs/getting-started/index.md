# Getting started

Trove is a personal event journal: capture typed facts from the sources in your
life, store them in one durable SQLite journal, and retrieve them conversationally
via MCP.

## Recommended path

**New here?** Start with **[Try in a day](./try-in-a-day.md)** — a guided
afternoon experiment using built-in shortcut types and MCP query. You will install
Trove, post six typed events with `curl`, and search them back from Cursor.
